Patilcraft — Built For Real Shopping

Open-source Django e-commerce platform — focused on speed, clarity, and a clean buying experience.

Who we are

Patilcraft is a student-driven e-commerce reference application implemented with Django, designed to demonstrate practical patterns for products, carts, orders, invoices, and user profiles. The project is ideal as a learning scaffold or as a starting point for production-ready stores.

Core Features

Product Catalog
Category filters, images, gallery and ratings.
Cart & Checkout
Coupon support, consolidated invoices and order management.
Invoices
Downloadable PDF invoices (xhtml2pdf fallback enabled on Windows).

Project & Tech Notes

  • Backend: Django (views, models, templates)
  • PDF generation: prefers WeasyPrint; uses xhtml2pdf+ReportLab as a cross-platform fallback
  • Media: `MEDIA_URL` and `MEDIA_ROOT` configured for product images
  • Auth: Django auth + django-allauth for social sign-in
